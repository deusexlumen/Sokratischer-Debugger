# SOKRATISCHER DEBUGGER V2.0
### EXTERNAL FRONTAL LOBE INTERFACE

> *"The unexamined code is not worth executing."*

## /// SYSTEM_OVERVIEW

The **Sokratischer Debugger** is a semantic analysis tool designed to radically deconstruct imprecise language. Acting as an external logic processor, it scans user input for "Semantic Voids"â€”abstract terms lacking physical or causal referents (e.g., "Energy," "Vibration," "Potential").

The system forces a "gap analysis" using the **L-M-V Protocol**:
1.  **[L] Localization:** Where does the phenomenon exist in physical space?
2.  **[M] Mechanics:** What is the causal chain? (A causes B).
3.  **[V] Validation:** How is the state measured?

Powered by Google's **Gemini 3 Pro** (Reasoning) and **Gemini 2.5 Flash** (TTS), this application provides a brutalist, distraction-free environment for rigorous intellectual honesty.

## /// CORE_FEATURES

*   **Real-time Void Detection:** Identifies hollow words and logical fallacies in text, audio, or uploaded documents.
*   **Logic Inspection Matrix:** A visual dashboard breaking down detected terms into their L-M-V components.
*   **Spiegel Intervention:** A Socratic response generated by the AI to challenge the user's perception of reality.
*   **Neural Audio Synthesis:** High-fidelity TTS (Text-to-Speech) using Gemini 2.5 Flash to vocalize the intervention.
*   **Cognitive Load Configuration:** Selectable difficulty modes ranging from "Simplified" (Beginner) to "Ruthless" (God Mode/No Mercy).
*   **Multi-Modal Input:**
    *   **Text Terminal:** Direct keyboard input.
    *   **Voice Stream:** Real-time microphone transcription.
    *   **Data Ingestion:** PDF/TXT/MD file analysis.

## /// OPERATIONAL_MANUAL

### 1. Boot Sequence
Upon loading, the system presents the **Configuration Matrix**:
*   **Language:** Select `DE` (German) or `EN` (English).
*   **Cognitive Density:**
    *   *SIMPLIFIED:* Gentle explanations.
    *   *BALANCED:* Standard conversational logic.
    *   *ACADEMIC:* High-precision terminology.
    *   *RUTHLESS:* Safety protocols disabled. Pure logic.

### 2. I/O Stream (Left Panel)
*   **Text Input:** Type your thesis or statement into the terminal.
*   **Mic Input:** Click `MIC` to transcribe spoken thoughts.
*   **File Load:** Upload a text-heavy document for analysis.
*   **Send:** Initiates the `RUN_DIAGNOSTIC` protocol.

### 3. Logic Core (Right Panel)
Once analysis is complete:
*   **Intervention Layer:** The system's response appears here. Specific words will be highlighted in **RED**.
*   **Void Inspection:** Click any highlighted word to populate the **Logic Inspector** at the bottom.
    *   View the specific *Logic Gap*.
    *   Review the *Socratic Questions* regarding Localization, Mechanics, and Validation.
*   **Audio Control:** The system will auto-play the response. Use the `STOP_AUDIO_FEED` button to halt playback.

*   **Styling:** Tailwind CSS (Brutalist Design System).

## /// INSTALLATION_PROTOCOL

1.  **Clone Repository:**
    ```bash
    git clone [repo_url]
    ```
   

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Configuration:**
    Create a `.env.local` file in the root directory:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```

4.  **Initiate System:**
    ```bash
    npm run dev
    ```
